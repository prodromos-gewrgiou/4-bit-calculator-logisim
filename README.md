# 4-bit Calculator built in Logisim Evolution
A basic 4 bit calculator built on logic circuits with Logisim Evolution, capable of simple arithmetic operations (addition, subtraction).


Something quite simple for my first logic circuit project, minimized usage of AI and work found online. Tried
to experiment and figure most of it out by myself, in order to learn and get a bit more familiar with the deep logic behind 
such projects, which may come at cost of efficiency, but that was not the goal of this project.

Features:
- Input system (Number encoder, reset, subtraction)
- Carry prediction system.
- Overflow indication
- Manually built 7-Segment Display (2 digits). Definitely re-invented the wheel on that one, but I was curious to know how it worked (Might be overcomplicated).

## Built With
- Logisim Evolution
